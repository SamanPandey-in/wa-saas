# Whatsapp Saas Built

This project is a SaaS (Software as a Service) application built using Node.js and Express.js. It provides a platform for users to manage their WhatsApp accounts and automate various tasks related to WhatsApp messaging.

## Features
- User authentication and authorization
- WhatsApp account management
- Message scheduling and automation
- Analytics and reporting

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/SamanPandey-in/wa-saas.git
   ```
2. Navigate to the project directory:
   ```bash
   cd wa-saas
   ```
3. Install dependencies:
   ```bash
   npm run setup
   ```
4. Start the development server:
   ```bash
    npm run dev
    ```
5. Access the application at `http://localhost:5173` and server at `http://localhost:5000`.
